# PRET-NG 
Инструмент для тестирования безопасности сетевый принтеров на основе PRET (Printer Exploitation Toolkit)

Ниже представлена максимально подробная документация к проекту «Printer Toolkit» — набору инструментов для обнаружения, сбора информации и взаимодействия с сетевыми принтерами через протокол PJL (Printer Job Language) на порту 9100.
Документация описывает оба файла (exploits.py и main.py) с разбором каждого класса, метода и ключевых технических решений. Она ориентирована на то, чтобы после долгого перерыва вы могли быстро восстановить контекст и логику работы.
Общее описание проекта

Проект представляет собой асинхронный инструмент на Python для работы с сетевыми принтерами, поддерживающими протокол PJL (обычно порт 9100). Основные возможности:

- Сканирование локальной сети — быстрое обнаружение устройств, слушающих порт 9100.

- Сбор информации — получение идентификационных данных, конфигурации, статуса и переменных принтера через PJL-команды INFO.

- Отправка сообщений на дисплей — изменение сообщения на экране принтера (команды RDYMSG и OPMSG) с возможностью восстановления исходного состояния.

-  Параллельная обработка — использование asyncio для конкурентного выполнения запросов, что значительно ускоряет сканирование и сбор данных.

Проект состоит из двух модулей:

- exploits.py — низкоуровневая работа с PJL: отправка команд, парсинг ответов, классы DeviceInfoGatherer и DisplayMessageSender.

- main.py — высокоуровневая логика: сетевое сканирование (NetworkScanner), менеджер эксплойтов (ExploitManager), объединяющий класс PrinterToolkit и примеры использования в main().

Структура файлов
## exploits.py

Содержит два основных класса:

- DeviceInfoGatherer — сбор информации об устройстве через PJL.

- DisplayMessageSender — отправка и управление сообщениями на дисплее принтера.

Оба класса используют асинхронные методы для соединения по TCP на порт 9100 и обмена PJL-командами.
## main.py

Содержит:

- NetworkScanner — сканирование сети на наличие открытого порта 9100.

-  ExploitManager — обёртка над классами из exploits.py для удобного использования.

-  PrinterToolkit — высокоуровневый API, объединяющий сканирование и эксплуатацию.

- main() — асинхронная точка входа с примерами вызовов.
